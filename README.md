# ceabed

Reusable code for personal projects.

## About

This project was started in order to centralize common code used throughout the author's personal projects. Starting new projects was becoming cumbersome as old code needed to be tracked down or ported from different languages. Creating a reusable library out of all this code seemed the most logical solution to the problem. It was decided the library would be written in C. There are a few reasons for this choice:
- To allow for easy integration with future projects regardless of the programming language in which they are written.
- To help the author become more accustomed to writing code in C.
- To make writing bindings for D simple.
- As a bit of a fun challenge!

As of now, the project has little to no documentation. Comments are scarce and not all code has been thoroughly tested. This project is shared in the hopes that someone might find it interesting or educational.
